BookStore MiniProject

โปรแกรมจัดเก็บข้อมูลร้านหนังสือรูปแบบใหม่

"BookStore MiniProject เป็น Project ขนาดเล็กที่เหมาะกับการใช้เก็บข้อมูลร้านหนังสือขนาดย่อม"

< วิธีการใช้โปรแกรม >


หน้าต่าง login


ส่วนหน้าต่างของการ login ในหน้านี้เราสามารถเลือก login โดย ใช้ชื่อ admin เข้าไปได้เลย


หน้าต่าง Register


หากเรายังไม่มีรหัส login เรานั้นก็สามารถ Register เพื่อเป็นการสมัครใหม่ได้


หน้าต่างหลัก


เมื่อเรานั้นได้ทำการ login เข้ามาแล้วเราก็จะเข้ามาสู่ในหน้าหลักโดยในหน้านี้เรานั้นจะสามารถเลือกใช้งานต่อไปได้เลย


หน้าจัดการข้อมูลลูกค้า


เมื่อเรานั้นได้เลือกหน้าจัดการข้อมูลลูกค้าแล้วเราสามารถ ใส่รหัสสมาชิกแล้วทำการกดค้นหาเพื่อค้นหาข้อมูลของลูกค้า และเมื่อทำการค้นหาเสร็จสิ้นแล้ว เรานั้นสามารถกดตรวจสอบข้อมูลเพื่อที่จะได้แก้ไขข้อมูลหรือลบในทางด้านซ้ายได้ นอกจากนี้ยังเพิ่มสมาชิกได้อีกเรื่อยๆ


หน้าจัดการข้อมูลหนังสือ


เมื่อเรานั้นได้เลือกหน้าจัดการข้อมูลหนังสือแล้วเราสามารถ ใส่รหัสหนังสือแล้วทำการกดค้นหาเพื่อค้นหาข้อมูลของหนังสือ และเมื่อทำการค้นหาเสร็จสิ้นแล้ว เรานั้นสามารถกดตรวจสอบข้อมูลเพื่อที่จะได้แก้ไขข้อมูลหรือลบในทางด้านซ้ายได้ นอกจากนี้ยังเพิ่มหนังสือได้อีกเรื่อยๆ


หน้าสั่งซื้อหนังสือ


เมื่อเรานั้นได้เลือกหน้าสั่งซื้อหนังสือแล้วให้เราทำการใส่รหัสหนังสือแล้วทำการกดเลือกเพื่อตรวจสอบข้อมูลอีกครั้ง หลังจากนั้นให้เราทำการเลือกจำนวนหนังสือตามต้องการ หลังจากนั้นให้เราทำการใส่รหัสสมาชิกของลูกค้า แล้วกดยืนยัน เพื่อทำการยืนยันการสั่งซื้อ หลังจากนั้นก็จะได้สรุปรายการออกมา


หน้าประวัติการซื้อ


ในหน้าประวัติการซื้อนี้จะทำการแสดงประวัติการซื้อทั้งหมดออกมานั่นเอง
